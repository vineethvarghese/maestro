PartialParser

case class PartialParser(fields: List[DateTimeFieldType], parser: InputParsers.Parser[DateTime]) extends Parser[DateTime] with Product with Serializable

Building block of an input file name parser.

Has a pre-determined list of date time fields it sets. Some primitive input parsers have nothing in their fields list. This is fine: it indicates the date times their parsers return are dummy values with no fields set.
